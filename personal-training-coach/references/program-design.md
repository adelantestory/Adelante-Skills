# Program Design Reference

## Program Type Selection — Decision Tree

### Step 1 — Available Days and Time
- 2 days/week → Full body (2x/week)
- 3 days/week → Full body (3x/week) or Push/Pull/Legs (1x each)
- 4 days/week → Upper/Lower split (2x each)
- 5–6 days/week → Push/Pull/Legs (2x each) or body part split

### Step 2 — Goal
- Muscle building / recomposition → Hypertrophy focus, 8–15 rep ranges
- Strength building → Strength focus, 3–6 rep ranges, compound movements primary
- Weight loss → Higher frequency, metabolic conditioning, moderate weights
- Endurance → Sport-specific cardio + supporting strength work
- Mobility → Corrective exercise, flexibility, low-impact movement
- General fitness → Balanced program across all qualities

### Step 3 — Equipment
See Equipment Configurations section below.

### Step 4 — Health Flags
Apply health flag modifications from SKILL.md before finalizing any program.

---

## Session Structure (Every Session)

1. Core activation — Dead bug + bird dog (when lower back flag is present — activates spinal stabilizers before loading)
2. Warm-up specific to primary movement
3. Compound movements — heaviest, most demanding first
4. Accessory movements — isolation, lighter, higher rep
5. Finishers (optional) — bands, bodyweight, conditioning
6. Mobility — relevant to the person's flags and goals

## Program Templates

**Note:** The templates below are resistance training structures for muscle building, recomposition, strength, weight loss, and general fitness goals. For endurance and mobility goals, program design is structured differently — see the Endurance and Mobility sections under Exercise Selection By Goal.

### Full Body 3x/Week (beginner to intermediate)
- Session A: Squat pattern + horizontal push + horizontal pull + core
- Session B: Hip hinge + vertical push + vertical pull + core
- Session C: Lunge pattern + compound push + compound pull + core

### Upper / Lower 4x/Week (intermediate)
- Upper A: Horizontal push primary + vertical pull primary
- Lower A: Squat primary + hamstring secondary
- Upper B: Vertical push primary + horizontal pull primary
- Lower B: Hip hinge primary + quad secondary

### Push / Pull / Legs 6x/Week (intermediate to advanced)
- Push A: Chest emphasis
- Pull A: Back width emphasis
- Legs A: Quad emphasis
- Push B: Shoulder emphasis
- Pull B: Back thickness emphasis
- Legs B: Posterior chain emphasis

### Minimalist 2x/Week (time-constrained)
- Session A: Lower body compound + upper push + core
- Session B: Lower body hinge + upper pull + cardio

### Endurance Program Structure (3–5 days/week)
- Primary sessions: Sport-specific cardio at prescribed zones (easy, moderate, threshold)
- Supporting sessions (1–2x/week): Single-leg strength, hip stability, core, injury prevention
- Structure varies significantly by sport — running, cycling, swimming, and rowing all have different periodization needs

### Mobility Program Structure (2–4 days/week)
- Each session: Joint mobility → flexibility → movement pattern practice → relaxation
- Progress by increasing range, improving control, and reducing pain — not by adding load
- Can be structured as dedicated sessions or added as 15–20 min before or after other training

## Rep Range and Intensity Guidelines

| Goal | Rep Range | Intensity (% 1RM) | Rest |
|---|---|---|---|
| Strength | 3–6 | 80–90% | 3–5 min |
| Hypertrophy | 8–12 | 67–80% | 60–90s |
| Definition / Recomp | 12–15 | 60–70% | 45–60s |
| Endurance | 15–20+ | 50–60% | 30–45s |

**Eccentric emphasis:** For hypertrophy, prescribe 2–4 count eccentrics on key exercises.

## Exercise Selection By Goal

### Muscle Building / Recomposition
Priority: Bench press, squat, Romanian deadlift, overhead press, pull-ups, rows, dips, curls
Emphasis: Full range of motion, controlled eccentrics, mind-muscle connection

### Strength Building
Priority: Conventional deadlift, barbell squat, bench press, overhead press, barbell row
Emphasis: Compound multi-joint movements, strength zone loading, technique first

### Weight Loss
Priority: Compound movements that burn the most calories
Add: Cardio conditioning circuits, metabolic finishers

### Endurance
Primary: Sport-specific cardio programming
Supporting strength: Single-leg exercises, hip stability, core, injury prevention

### Mobility and Health
Priority: Hip flexor stretches, thoracic rotation, pigeon pose, cat-cow, bird dog, dead bug, glute bridges, band work

### Lower Back Protected Program
Remove: Conventional deadlift, bent-over barbell rows, good mornings, heavy back squats
Replace: Romanian DL, chest-supported rows, single-arm DB row with knee support, heel-elevated squat, leg press

## Equipment Configurations

### Full Barbell + Rack + Dumbbells
Full program available. Use PPL or Upper/Lower split.

### Dumbbells Only
- Replace barbell bench → DB bench or floor press
- Replace barbell squat → DB goblet squat, DB Bulgarian split squat
- Replace barbell deadlift → DB Romanian DL, single-leg DB deadlift
- Replace barbell row → DB bent-over row, single-arm DB row
- Upper/Lower or Full Body split recommended

### Resistance Bands Only
Full body program, higher rep ranges. Suitable for mobility and general fitness. Strength building limited — manage expectations.

### Pull-Up Bar + Bands + Dumbbells
Effective muscle building and recomposition programs achievable with creative substitutions.

### Bodyweight Only
Full body 3x/week. Progressive overload through: increasing reps → harder variation → tempo → reduced rest → plyometric variation.

### Hotel Gym (dumbbells + cardio only)
Pull and upper body sessions work well. Legs: DB Romanian DL, goblet squat, walking lunges. No barbell sessions.
